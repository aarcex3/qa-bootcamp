# qa-bootcamp-homework

## Develoment
Clone the repo into your ide.\
Tasks are in the Main class, uncomment them to try them in the terminal.\
Run `mvn test` to run the tests

### Requirements
- Java 1.8
- Maven
- JUnit 5
- [Extension pack for java](https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack) if using vscode
